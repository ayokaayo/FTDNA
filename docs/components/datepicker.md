# Datepicker

The `FTDatepicker` component provides a user-friendly interface for selecting dates. It supports single date selection, date ranges, and various formatting options.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTDatepicker } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTDatepicker } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTDatepicker }
  }
</script>
```
