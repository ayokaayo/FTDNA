# Floating Label

The `FTFloatingLabel` component provides animated label behavior for form inputs. The label floats above the input when focused or when the input has a value.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTFloatingLabel } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTFloatingLabel } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTFloatingLabel }
  }
</script>
```
