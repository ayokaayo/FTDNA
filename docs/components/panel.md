# Panel

The `FTPanel` component is a container for organizing content with headers, footers, and flexible layouts. It supports collapsible behavior and various visual styles.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTPanel } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTPanel } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTPanel }
  }
</script>
```
