# Table

The `FTTable` component displays data in a structured tabular format with support for sorting, filtering, pagination, and various column types.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTTable } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTTable } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTTable }
  }
</script>
```
