# Tabs

The `FTTabs` component organizes content into multiple sections, each accessible through tab buttons. Users can switch between tabs to view different content.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTTabs } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTTabs } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTTabs }
  }
</script>
```
