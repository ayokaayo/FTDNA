# Input

The `FTInput` component is a versatile text input field with support for various input types, validation states, placeholders, and disabled states. It provides v-model binding and event handling.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTInput } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTInput } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTInput }
  }
</script>
```
