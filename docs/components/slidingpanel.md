# Sliding Panel

The `FTSlidingPanel` component displays content in a panel that slides in from the side of the screen. It supports animations, configurable width, and event handling for open/close states.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTSlidingPanel } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTSlidingPanel } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTSlidingPanel }
  }
</script>
```
