# Confirm

The `FTConfirm` component displays a confirmation dialog to users, typically used for confirming destructive actions or important decisions.

## Usage

### Composition api (script setup)

```vue
<script setup lang="ts">
  import { FTConfirm } from '@fasttrack-solutions/vue-components-lib';
</script>
```

### Options api

```vue
<script>
  import { FTConfirm } from '@fasttrack-solutions/vue-components-lib';
  export default {
    components: { FTConfirm }
  }
</script>
```
