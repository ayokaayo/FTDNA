# Typography

FastTrack uses **Inter** for all UI text and **Noto Sans Mono** for code. Two weights are available: Regular (400) and Bold (700). The type scale has **17 styles** across 5 categories.

## Usage

```css
.heading {
  font-family: var(--ft-font-family-text);
  font-weight: var(--ft-font-weight-bold);
}
```

```vue
<template>
  <h2 style="font-family: var(--ft-font-family-text); font-size: 20px; font-weight: 700; line-height: 24px;">
    heading-m-bold
  </h2>
</template>
```
