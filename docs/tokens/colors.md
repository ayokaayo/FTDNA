# Colors

FastTrack's color system is aligned to the Figma Brand Colours variable library. All token numbers match Figma variable names directly (e.g., Figma `pink/400` = token `--ft-color-brand-pink-400`). Colors are available as CSS custom properties (`--ft-color-*`), SCSS variables (`$ft-color-*`), and TypeScript constants.

## Usage

```css
/* Use CSS custom properties */
.button-primary {
  background: var(--ft-color-primary-base);
  color: var(--ft-color-on-primary-base);
}
```

```scss
// Use SCSS variables
.button-primary {
  background: $ft-color-primary-base;
  color: $ft-color-on-primary-base;
}
```

```ts
// Use in Vue components
import { FT_COLOR_PRIMARY_BASE } from '@fasttrack-solutions/design-system/dist/useTokens'
```
